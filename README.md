## ✨ Özellikler

### 📚 Kapsamlı Soru Arşivi
- **2000+** güncel sınav sorusu
- Grundstoff (Temel bilgiler) soruları
- Klasse B, C ve CE sınıflarına özel sorular
- Görselli, video ve sayısal soru tipleri

### 📝 Gerçekçi Sınav Simülasyonu
- 30 soruluk deneme sınavları
- Gerçek sınav puanlama sistemi
- Hata puanı hesaplama
- Anlık sonuç değerlendirmesi

### 🎯 Pratik Mod
- Tüm soruları tek tek inceleme
- Cevap kontrolü ve açıklamalar
- Kendi hızınızda öğrenme
- İlerleme takibi

### 🔍 Akıllı Arama & Filtreleme
- Soru koduna göre arama
- İçeriğe göre arama
- Kategori bazlı filtreleme
- Sınıf bazlı ayrım (B, C, CE)

### 📱 100% Mobil Uyumlu
- **Responsive tasarım** - Her ekran boyutunda mükemmel çalışır
- Telefon, tablet ve bilgisayar desteği
- Dokunmatik ekran optimize
- Hızlı ve akıcı performans
- Çevrimdışı çalışma desteği (PWA hazır)

### 🌍 Çok Dilli Destek
- 🇹🇷 Türkçe
- 🇩🇪 Deutsch (Almanca)
- Dil tercihi kaydetme

## 🚀 Kullanım

### Web Sitesinden
1. [https://ubden.github.io/fahrschule-web/](https://ubden.github.io/fahrschule-web/) adresine gidin
2. **Pratik Yap** veya **Deneme Sınavı Yap** butonlarını kullanın
3. İstediğiniz dilde öğrenmeye başlayın!

### Yerel Olarak Çalıştırma
```bash
# Repoyu klonlayın
git clone https://github.com/ubden/fahrschule-web.git

# Klasöre girin
cd fahrschule-web

# Basit bir HTTP sunucusu başlatın
python -m http.server 8000
# veya
npx serve

# Tarayıcınızda açın
# http://localhost:8000
```

### ⚠️ Sorumluluk Reddi & Uygun Kullanım

**BU PROJE BIR EĞİTİM ARACI VE YARDIMCI KAYNAKTIR:**

1. **Resmi Kaynak Değildir**
   - Bu platform resmi Almanya ehliyet sınavı kurumu tarafından sağlanmamaktadır
   - Sorular eğitim ve pratik amaçlıdır
   - Resmi sınav soruları farklılık gösterebilir

2. **Garanti Verilmez**
   - İçeriğin doğruluğu için elimizden geleni yapsak da **%100 garanti veremeyiz**
   - Güncellemeler ve değişiklikler zaman alabilir
   - Sınav başarısı garantisi yoktur

3. **Uygun Kullanım**
   - Bu kaynak **yasal ve etik** şekilde kullanılmalıdır
   - Sınav sırasında **kopya** veya **hile** amacıyla kullanılamaz
   - Sadece **hazırlık ve öğrenme** amaçlı kullanılmalıdır
   - Gerçek sınavda bu kaynaklara erişim **yasaktır**

4. **Sorumluluk**
   - Kullanıcılar bu kaynağı **kendi sorumluluklarında** kullanır
   - Sınav başarısızlığından veya yanlış bilgilerden dolayı **sorumluluk kabul edilmez**
   - Resmi sürücü kursu ve eğitmen desteği alınması **şiddetle tavsiye edilir**

5. **Ticari Kullanım Yasağı**
   - Bu içerik **üçüncü kişiler** tarafından **ticari amaçla kullanılamaz**
   - Satış, kiralama veya ücretli hizmet olarak sunulamaz
